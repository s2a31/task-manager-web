About the Task Manager Application

This Task Manager, created by Szabolcs Balogh, is more than just a coding project; it is a demonstration of practical skill in building dynamic web applications. Developed using Flask, a powerful micro web framework in Python, it highlights my understanding of both front-end and back-end development.

Core Features

  • User Authentication: Utilizing Flask-Bcrypt for secure password hashing, the application ensures safe user registration and login processes.
  • Database Integration: By integrating MySQL, a robust and scalable database, it efficiently handles user and task data.
  • Task Management: Users can add, update, and delete tasks, each with attributes like name, description, due date, and status.
  • Interactive UI: The front-end design, enhanced with Bootstrap and custom CSS, offers a responsive and user-friendly interface.
  • JavaScript Enhancements: The use of JavaScript for features like drag-and-drop functionality and real-time updates enhances user experience.

Technical Aspects

  • Flask Session Management: Ensures a secure and interactive user experience.
  • MySQL Database: A powerful and efficient choice for data storage, providing robust management of user and task information.
  • Responsive Design: Ensures compatibility with various devices, enhancing accessibility.
  • Security Measures: Flask's secure session handling, coupled with bcrypt for password hashing, adds robust security layers.

Learning Journey

This project is a milestone in my journey of transitioning from tourism to IT. It reflects my growth in the IT field and my commitment to continuous learning through platforms like Udemy, Pluralsight, and CS50.
